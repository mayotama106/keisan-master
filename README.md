# めざせ、％マスター！ 📱💯

隙間時間に **％（パーセント）計算** を鍛える、**オフライン対応のスマホ向け学習アプリ（PWA）**です。
割引・○％増し（マージン）・「A は B の何％？」までを、電卓なしでスッと解けるように。
四則演算も「基礎練習」として残しています。設計書（[設計書.md](設計書.md)）がベースです。

- ✅ **インストール不要・ビルド不要** — ブラウザだけで動きます
- ✅ **オフライン動作** — 一度開けば通信なしで遊べます（PWA / Service Worker）
- ✅ **データは端末内のみ** — 個人情報なし・外部送信なし（localStorage）
- ✅ **ホーム画面に追加** すれば、ふつうのアプリのように起動できます

---

## 🚀 すぐに試す（ローカルで動かす）

PWA（インストール・オフライン）を正しく動かすには、**簡易サーバー経由**で開くのがおすすめです。

### macOS の方（いちばん簡単）
1. このフォルダ内の **`start-mac.command`** を**ダブルクリック**
2. 自動でローカルサーバーが立ち上がり、ブラウザで開きます
3. 終了するときは開いたターミナルウィンドウを閉じてください

### 手動で起動する（Mac / Windows / Linux 共通）
このフォルダで以下を実行し、表示された URL をブラウザで開きます。

```bash
# Python がある場合（macOS は標準で入っています）
python3 -m http.server 8000

# Node.js がある場合
npx serve .
```

→ ブラウザで http://localhost:8000 を開く

> **ファイルを直接ダブルクリック（`index.html`）でも遊べます**が、その場合オフライン用の
> Service Worker は動きません（アプリ自体は問題なく動作します）。

## 📲 スマホにインストールする方法（人に共有する）

「人に渡して、その人のスマホで使ってもらう」場合は、**URL を共有するのが最も簡単**です。

### 方法A：無料の静的ホスティングに置いて URL を配る（おすすめ）
このフォルダ一式をそのままアップロードするだけで動きます。ビルド不要です。

- **Netlify Drop**: <https://app.netlify.com/drop> にフォルダをドラッグ＆ドロップ → 即 URL 発行
- **GitHub Pages**: リポジトリに push → Settings → Pages で公開
- **Vercel / Cloudflare Pages** などでも同様（フレームワーク設定は「None / 静的」）

発行された URL をスマホで開き、

- **iPhone (Safari)**: 共有ボタン → 「ホーム画面に追加」
- **Android (Chrome)**: メニュー → 「アプリをインストール」/「ホーム画面に追加」

すると、アイコンから**普通のアプリのように**起動でき、オフラインでも動きます。

### 方法B：フォルダ（zip）をそのまま渡す
- `めざせ計算マスター.zip`（同梱）を渡し、受け取った人が解凍して
  上記「ローカルで動かす」の手順、または `index.html` を開くだけでも遊べます。

## 🎮 機能

| 機能 | 内容 |
|---|---|
| ％にチャレンジ | ％計算をランダムに連続出題 |
| ％ジャンル | 基本の％ / 割引 / ％増し / 「何％？」 |
| 基礎（四則） | 足し算 / 引き算 / 掛け算 / 割り算 も練習可能 |
| 📖 ％の使い方 | 小学生にもわかる、図つきの段階レクチャー |
| 💬 先生にきく | ％専門の「先生ボット」にチャットで質問（完全オフライン） |
| 即時採点 | テンキー入力 → 送信ですぐ正誤・とき方ヒント表示 |
| 復習 | 間違えた問題を自動記録、連続正解で「卒業」 |
| XP・レベル | 正解・速解・コンボでXP、レベルアップ演出 |
| 統計 | ジャンル別正答率・連続日数（ストリーク）など |
| 設定 | 難易度（やさしい/ふつう/むずかしい）・問題数・サウンド |
| バックアップ | 進捗を JSON でエクスポート / インポート |

### 💬 「先生にきく」チャットについて
- **内蔵の先生ボット**：ダウンロード不要・完全オフライン・iPhone でも軽快。
  「80の15%は？」のように聞くと手順つきで回答し、「割引って何？」等の概念質問にも答えます。
  - 大きな AI モデルは読み込まないので、起動が速く・通信不要で・電池にもやさしい設計です。

## 🛠 技術構成（ビルドなし・依存ゼロ）

純粋な HTML + CSS + バニラ JavaScript で実装しています。配布と「すぐ動く」を最優先し、
ビルドツールや npm パッケージは使っていません。

```
めざせ計算マスター/
├── index.html              # アプリ本体（エントリ）
├── manifest.webmanifest    # PWA 設定
├── sw.js                   # Service Worker（オフラインキャッシュ）
├── css/styles.css          # デザインシステム（設計書 §3 準拠）
├── js/
│   ├── storage.js          # localStorage 永続化
│   ├── generator.js        # 出題エンジン（％特化＋四則。プラグイン的に追加可能）
│   ├── xp.js               # XP・レベル計算
│   ├── sound.js            # WebAudio 効果音
│   ├── store.js            # 状態・採点・復習・統計の中枢
│   ├── ui.js               # DOM ヘルパ
│   ├── tutor.js            # ％の先生ボット（ルールベース・完全オフライン）
│   └── app.js              # 画面・遷移・出題・レクチャー・チャット
├── icons/                  # アプリアイコン（PNG / SVG）
├── start-mac.command       # macOS 用ワンクリック起動
└── 設計書.md               # 元の設計書
```

### ジャンルの追加（拡張）
[js/generator.js](js/generator.js) の `GENRES` 配列に1要素追加するだけで新ジャンルを増やせます。
出題ロジックは端末内の純粋関数なので、サーバーは不要です。

## ⚠️ 注意

- データはブラウザの localStorage に保存されます。**ブラウザのデータを消すと進捗も消えます**。
  大事な進捗は設定画面の「エクスポート」でバックアップできます。
- オンライン用のフォント（Poppins / Noto Sans JP）は、オフライン時は端末標準フォントに自動で切り替わります。
